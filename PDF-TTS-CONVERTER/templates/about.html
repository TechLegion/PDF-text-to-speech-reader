About This Project

This project, PDF to Text Speech Converter, is developed as part of the final year requirements for the Bachelor of Science degree in Computer Science at Redeemer's University. The primary aim of this project is to create a tool that converts PDF documents into spoken text, enhancing accessibility and usability for individuals with visual impairments or those who prefer auditory learning.

The project is supervised by [Supervisor's Name], a distinguished professor in the Department of Computer Science at [University Name]. This tool leverages advanced text-to-speech technology to provide a seamless and user-friendly experience for converting written content into natural-sounding speech.

Key features of this project include:

  • Easy-to-use interface for uploading and managing PDF files.
  • High-quality text-to-speech conversion with adjustable speech parameters.
  • Support for multiple languages and accents.
  • Downloadable audio files for offline use.

The development of this project involved extensive research and implementation of various software engineering principles, ensuring a robust and scalable solution. We hope this tool serves as a valuable resource for those in need of text-to-speech conversion capabilities.
